====================================================================
  📌 SAM Scheduler (Seedcore Scheduler) 사내 서버 관리자 가이드
====================================================================

1. 디렉터리 구성 안내
--------------------------------------------------------------------
  sp-server.exe         : 메인 웹 서버 (웹 UI + REST API + DB 통합 단일 실행 파일)
  sp-backup.exe         : 관리자 전용 DB 백업 및 특정 시점 복구 CLI 도구
  sp-reset-admin.exe    : 관리자(admin) 비밀번호 즉시 재설정 CLI 도구
  query_engine-*.node   : Prisma 데이터베이스 엔진 바이너리 (필수 유지)
  README.txt            : 본 사용 설명서 문서

  [서버 구동 시 자동 생성되는 폴더]
  ./data/               : SQLite 데이터베이스 파일 (sam.db) 보관 폴더
  ./logs/               : 일별 회전 로그 파일 (sp-YYYY-MM-DD.log) 보관 폴더 (최근 2주/14일 자동 보관 및 경과 파일 자동 삭제)
  ./backups/            : DB 백업 스냅샷 파일 저장 폴더

--------------------------------------------------------------------
2. 서버 실행 방법 & 사내 네트워크(LAN) 접속
--------------------------------------------------------------------
  1) sp-server.exe 를 더블 클릭하여 실행합니다.
  2) 최초 실행 시 데이터베이스(./data/sam.db)가 자동 초기화되고 웹 서버(3000번 포트)가 동작합니다.
  3) 콘솔 창에 표시되는 IP 주소로 브라우저(Chrome/Edge 등)에서 접속합니다:
     - 호스트 PC 본인 접속 : http://localhost:3000
     - 사내 타 PC 접속     : http://<콘솔에표시된IP>:3000
  4) 사내 타 PC에서 접속이 안 될 경우 윈도우 방화벽 인바운드 규칙에서 TCP 3000번 포트를 허용해 주세요.

--------------------------------------------------------------------
3. 기본 관리자 계정 정보
--------------------------------------------------------------------
  - 아이디   : admin
  - 비밀번호 : ChangeMe!Now
  * 최초 로그인 시 비밀번호 변경 화면으로 이동합니다.

--------------------------------------------------------------------
4. 관리자 보조 도구 사용 설명서
--------------------------------------------------------------------
[A] DB 백업 및 특정 시점 복구 (sp-backup.exe)
    - 수동 백업 수행 (backups/ 폴더에 sam_YYYYMMDD_HHMMSS.db 스냅샷 생성):
      > sp-backup.exe backup

    - 백업 목록 확인:
      > sp-backup.exe list

    - 특정 날짜 백업 파일로 DB 복구 (복구 전 현재 DB는 자동 안전 백업됨):
      > sp-backup.exe restore sam_20260723_120000.db
      * 복구 후 sp-server.exe를 재시작해 주세요.

[B] 관리자 비밀번호 재설정 (sp-reset-admin.exe)
    - sp-server.exe 가 실행 중일 때 (또는 최초 구동되어 DB가 생성된 후) 동작합니다.
    - 비밀번호 분실 시 원하는 새 암호로 즉시 변경 및 계정 잠금 해제:
      > sp-reset-admin.exe 원하는새비밀번호123!
    * sp-server.exe 가 실행 중인 상태에서 즉시 변경되며, 서버 재시작 없이 새로 지정한 비밀번호로 바로 로그인 가능합니다.
